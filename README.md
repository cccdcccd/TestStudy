# 解答入力アプリ（Answer Sheet App）

勉強中に使用するデジタル解答用紙アプリ。PC・スマホ両対応のシングルページWebアプリ。

🌐 **公開URL:** https://cccdcccd.github.io/TestStudy/

---

## 成果物ファイル

| ファイル | 役割 |
|---|---|
| `index.html` | アプリ本体・UIの骨格 |
| `style.css` | デザインシステム（CSS Variables・ダークモード・レスポンシブ） |
| `app.js` | フォームビルダー・解答入力・セッション管理のメインロジック |
| `storage.js` | LocalStorage による永続化（フォーマット・回答セッション） |

---

## 開発工程と実装内容

### フェーズ1：基盤構築
- `index.html` / `style.css` / `app.js` / `storage.js` の4ファイル構成を設計
- ダークモードベースのデザインシステムを CSS Variables で実装
- 2タブ構成（フォーマット作成 / 解答入力）の骨格を作成

### フェーズ2：バグ修正
- `type="module"` を使った ES モジュールが `file://` プロトコルで CORS エラーになる問題を修正
- `import/export` を廃止し、通常の `<script>` タグ2枚構成（`storage.js` → `app.js`）に変更

### フェーズ3：回答タイプ自動生成
- 選択肢を手動で1件ずつ追加する手間を廃止
- 「回答タイプ」と「個数」を選ぶと選択肢を自動生成するUIに刷新

| タイプ | 生成内容 | 個数指定 |
|---|---|---|
| 数字 | 1, 2, 3… | あり |
| カタカナ | ア, イ, ウ… | あり |
| アルファベット | A, B, C… | あり |
| ○✕ | ○, ✕ | なし（固定） |
| 記述 | テキストエリア | なし |

### フェーズ4：OKボタンで次問自動追加
- 各問題カードに「OK — 次の設問へ ▶」ボタンを追加
- OKを押すと同じ設定の次問を直下に自動追加してスクロール＆フォーカス

### フェーズ5：タップ式回答UI
- プルダウン（`<select>`）を廃止
- 選択肢を全てボタンとして一覧表示し、タップ1つで回答できるUIに変更
- 選択済みはアクセントカラーでハイライト、`×` ボタンで解除

### フェーズ6：複数回答セッションの保存・呼び出し
- 回答を任意の名前で複数保存できる機能を実装
- 保存時にスキーマスナップショット（問題構成）も同時保存
- 3タブ目「📚 保存済み回答」でフォーマット名ごとにグループ表示
- 「呼び出し」でフォーマット＋回答を完全復元して解答タブへ自動切り替え

### フェーズ7：UI改善
- 「問題を追加」ボタンをリスト下部に移動
- 初期表示から1問目を表示済みにして即座に作業開始できるよう変更

### フェーズ8：GitHub Pages 公開
- `git init` → `git commit` → `git push` を実施
- GitHub Pages（Deploy from branch: `main`）で静的サイトとして公開
- スマホブラウザからURLへアクセスするだけで利用可能

---

## 主な機能

- **フォーマット作成:** 問題を任意の順番・タイプで構成。LocalStorageに名前付きで保存・読み込み・削除
- **解答入力:** タップ式チップボタンで素早く回答。記述式はテキストエリア
- **一括クリア:** 全回答をワンタップでリセット
- **回答の保存・呼び出し:** セッション名を付けて複数保存。後から呼び出して確認・修正可能

---

## 動作環境

- **PC:** Chrome / Edge / Firefox（`file://` でダブルクリック起動も可）
- **スマホ:** Safari（iOS）/ Chrome（Android）でURL直接アクセス
- **データ:** LocalStorage に保存（サーバー不要・クラウド同期なし）
